# Data Cleaning and Formatting (Excel)

## Overview
This project demonstrates data cleaning and formatting using Excel / Google Sheets.

## Tools Used
- Google Sheets
- Microsoft Excel

## Steps Performed
- Imported raw CSV data
- Identified and handled missing values
- Checked and validated duplicate records
- Standardized text and formatting
- Prepared clean, analysis-ready dataset
